Programs and Games
3D graphics

Alcounter

My android app for keeping track of blood alcohol content while drinking, and not after like with other similar apps. Partly made in one school project, and then finished in my free time. Made mainly to learn app making and publishing on Google Play Store.

Slušovice map for BeamNG.drive

Map for the BeamNG.drive racing simulator game. I wanted to make part of horse race track, that is near where I live, and is used as rally stage few times a year. I also wanted to learn 3D moddeling, texturing and using map editor.

Still work in progress, no download available yet.

briskGaugeAC

My app for Assetto Corsa. Displays essential information in Škoda fabia lookalike instrument cluster. I made this because I wanted to learn how apps in Assetto Corsa are made, and I did not like any of the existing instrument cluster apps.

Check it out!

resizeAndWatermark

My plugin for GIMP that can add "watermark" to selected coordinates, resize the image and export it. I made it because I wanted to learn how to make plugins for GIMP. I use it when I edit photos and want to add watermark on the same spot and resize and export multiple photos at the same time.

Check it out!

Robot

I designed and 3D modelled this robot in Blender. Design is based on Mr. Handy from Fallout series, and from cars and jet planes from the 60s. The design and model was made as an assignment for 3D character modelling on FI MU.
The model is rendered using blender cycles and HDRI world texture.

Link to the HDRI

Go Kart

I 3D modelled this kart in Blender. Design is based on some real life go karts. The model is stylized to the "low poly" look, but can be made more smooth by subdivision. This model will be used in a game I am working on.
The model is rendered using blender cycles and HDRI world texture.

Wild Boar

I 3D modelled this in Blender. Design is based on the stuffed animal trophies. I wanted to play around with blender fur and particles, so I chose to model a pig. The model was made as an assignment for 3D character modelling on FI MU.
The model is rendered using blender cycles and HDRI world texture. I played around with colored lights for the render.

Link to the HDRI
